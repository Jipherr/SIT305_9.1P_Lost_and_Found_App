# Simple Lost and Found App
This app allows you to create, view, filter, and remove lost and found adverts. You can use it by clicking the "Create a New Advert" button on the main screen, filling in the required information (including picking a date and uploading a photo), then clicking the "Save Advert" button. The advert will now show on the "Show All Items" page. You can filter the list by category using the dropdown menu at the top. Click on any item card to view its full details, or click the "Remove Advert" button on the details page to delete the advert once the item is returned.

Regarding Image Uploads:
You will notice that while an image upload is strictly required to create a new advert, the image is not displayed on the final Item Details screen. 
This is because of two reasons, the assessment outline and provided wireframes did not mandate that the image had to visually rendered on the details page and the app kept crashing when I had tried to add the image to the details page.

This app was created for SIT305 Task 7.1 Pass Task.
