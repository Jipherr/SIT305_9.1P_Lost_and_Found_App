# Simple Lost and Found App with Map
This app allows you to create, view, filter, map, and remove lost and found adverts. You can use it by clicking the "Create a New Advert" button on the main screen, filling in the required information (including picking a date, uploading a photo, and using the "Get Current Location" button to automatically fetch your street address), then clicking the "Save Advert" button. The advert will now show on the "Show All Items" page. You can filter the list by category using the dropdown menu at the top. Click on any item card in the list to view its full details, or click the "Remove Advert" button on the details page to delete the advert once the item is returned. Additionally, you can click the "Show On Map" button on the main screen to view all lost and found items plotted on a Google Map, which includes a slider to filter and display items within a specific radius of your current location. 

This app was created for SIT305 Task 9.1 Pass Task.
